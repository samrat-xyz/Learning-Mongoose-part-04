# Getting Started with Mongoose (Lesson 04)


This project is a simple **Express + Mongoose** backend application created for practicing MongoDB database connection and basic user creation using REST APIs.

---

## Features

- Express server setup
- MongoDB connection using Mongoose
- Environment variables with dotenv
- User model with validation
- Create user API (POST)
- CORS & JSON middleware enabled

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- cors
